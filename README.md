# Wox.Plugin.Stock
Stock plugin for Wox-launcher

## Install
Install requests & Beautiful Soup with shell.
```console
$ pip install requests
✔ Installation Succeeded
$ pip install beautifulsoup4
✔ Installation Succeeded
…
```
Install Stock with Wox.
```
wpm install Stock
```
